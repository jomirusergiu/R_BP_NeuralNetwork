R_ParallaxMotion

R_ParallaxMotion is a small UIView category for parallax effects based on UIMotionEffect.

Usage

 view.parallaxValue = 50;

You can set motion amplitude by changing the value. Positive values make the view appear to extend from the screen. Negative values make the view appear to recess behind the screen.
